For this project I wanted to keep messing around with the rotating spheres from my norm project last week but also trying to create more of a unified concept. I ended up taking a lot of inspiration from the eye of sauron at first, but wasn't really happy with how the textures wrapped around the eyeball 3d model I added, so decided to make a burning kiln where I changed the cubes and spheres into glass chunks that were getting tossed around within the fire. To add an element of interactivity, I made it so some of the squares randomly shift positions on mouse click. I wanted to emphasize the eyeball model while also taking advantage of the fact that you can see inside the model partially from certain angles to create a unique perspective.

For tools, I used the free version of copilot to check for any mistakes within my writing and for potential inconsistencies in placement I might have missed before submitting, and used the class resources while also looking up additional resources to try to smooth out my textures and model a bit since I wasn't really happy with how the model looked initially, and wanted to see if I could stretch the model and recolor it as wanted to see what other options I had availiable. I've found these recent projects focused on 3d models and shapes to be really interesting, and I'd love to try my hand and making my own next.
